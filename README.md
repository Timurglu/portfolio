# Timur Gluhov Portfolio

Static portfolio site prepared for GitHub Pages.

## Structure

- `index.html` — main page
- `silkway.html` — Silkway case study
- `case-02.html` — placeholder case page
- `case-03.html` — placeholder case page
- `styles.css` — styles
- `script.js` — reveal animations
- `assets/` — images and case assets with GitHub-safe file names

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload the project files from this folder.
3. In repository settings, open `Pages`.
4. Set source to `Deploy from a branch`.
5. Select the main branch and the root folder `/`.

After that, GitHub Pages will serve `index.html` automatically.
